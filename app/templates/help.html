Help


Welcome to Volunteer Tutor Exchange. This introduction explains how the system works and how to use the site.

1. Account types and user roles

When you signed up with VTE, you created either a student or a tutor account. Tutor accounts are 'upgraded' student accounts. A tutor account can do anything a student account can do. Users with tutor accounts can also have their contact details searched and can create session requests.

Tutorial sessions involve two users. One user takes the tutor role and the other takes the student role. Users with student accounts can only take a student role in a tutoring session. Tutors can take either role (ie. tutors can be tutored by other tutors). For this introduction, it is assumed that the user taking the student role has a student account.

2. User profile page

Users can see and update their profiles by clicking on the profile tab. Please try to keep your profile up to date. Some fields cannot be edited through the website. Please contact an administrator to change these fields.

Important note for tutors:

The Phone, Units, Languages and Bio fields can be seen by students through the search feature. A bio is not required to sign up and will be blank until you submit one. You can toggle search visibility to make your profile unsearchable.

3. Finding a tutor / the search page

Students who wish to organise a tutoring session with a tutor must contact a tutor through either the phone number given or student email. These can be found through the search page. Search for a unit by typing the unit code into the search bar, selecting the appropriate unit and clicking on the search button. A list of tutors will be return. Click on a name to see details.

4. Tutoring session requests

As mentioned above, students initiate scheduling a tutoring session by contacting a tutor. After a date and time has been agreed on, the session must be registered with the system in a two step process:

  1. The tutor sends a session request by clicking on the 'session request' button on the sessions page (button only visable to tutors). He / she fills out the form and clicks submit.
  2. The student needs to accept the session. A new entry will be created in the session requests table (may have to press refresh at bottom of page to see it). If the date and time are correct, press the green accept button. Otherwise press the red reject button and contact tutor to send another session request. Be sure to accept the request before the hour-long session has ended.

Pro tip: If contacting tutor by phone, send and accept session request before ending the call.

5. Canceling and rescheduling tutoring sessions

Tutoring sessions can be canceled or rescheduled by the tutor. Students should contact their tutor if a session needs to be cancelled or rescheduled.

A list of upcoming session is displayed in the upcoming session table. Tutors can cancel/reschedule a session by clicking on the respective button at the end of the row of the table (only visable to tutors). Rescheduling a session will result in sending a new session request.

6. Acknowledging completed sessions

In order for a tutor to receive accredited volunteer hours, both tutor and student must acknowledge the completion of the session.

One day after a tutoring session has been completed, the session will move from the upcoming session table to the Session for sign off table. To affirm that the session has been completed, click the green tick button. If a session was not completed, click the red cross button and enter the reason the session was not completed.

All negative responses will be reviewed by a VTE team member before volunteer hours are discredited.
